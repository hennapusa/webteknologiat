First chapter

The CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning. The grid properties are supported in all modern browsers. A grid layout consists of a parent element, with one or more child elements. The lines between columns are called column lines. The lines between rows are called row lines.

Second chapter

The grid-template-columns property defines the number of columns in your grid layout, and it can define the width of each column. The value is a space-separated-list, where each value defines the width of the respective column. If you want your grid layout to contain 4 columns, specify the width of the 4 columns, or "auto" if all columns should have the same width.

Third chapter

By default, a container has one grid item for each column, in each row, but you can style the grid items so that they will span multiple columns and/or rows. The Grid Layout allows us to position the items anywhere we like.
